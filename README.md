### Prerequisite
* **Python Runtime:** Python versi 3.8 atau yang lebih baru.

### Cara Menjalankan Program
Lakukan _clone_ terhadap repository:
```bash
git clone https://github.com/rafenmaxxx/UAS-STBI.git
```

Ubah direktori pada terminal:
```bash
cd UAS-STBI
```

Jalankan program `lambda.py` untuk pengujian pengaruh parameter bobot:
```bash
python lambda.py
```

Jalankan program `querylog.py` untuk demonstrasi simulasi pengaruh _query log_ pengguna:
```bash
python querylog.py
```
